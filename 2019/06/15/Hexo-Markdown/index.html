Markdown格式

首行缩进

&ensp; //相当于1个空格  
&emsp; //相当于2个空格，1个汉字

分段: 两个回车
换行: 两个空格 + 回车
标题: # ~ ######，#号的个数表示几级标题，即表示一级标题到六级标题

强调

引用: > 注意后面紧跟个空格

以下为引用效果
继续引用

二级引用

表格: - 和 | 分割行和列 ， : 控制对齐方式

0 1 2
0 1 2
0 1 2
0 1 2

代码块: 四个空格开头或三个` 链接: [文字](链接地址)
图片: ![图片说明](图片地址)，地址可以是本地路径，也可以是网络地址
列表: * ， + ， - ， 1. ，选其中之一，注意后面紧跟个空格
